Exercise Tracker

1) Create your username
2) Get your unique _id in a JSON response
3) Use the _id as your userId
4) Add your exercise to the database

Create a New User

POST /api/exercise/new-user

Add Exercises

POST /api/exercise/add

GET your exercise log: GET /api/exercise/log?{userId}[&from][&to][&limit]

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

Example: https://exercise-tracker-project.jylee3.repl.co/api/exercise/log?userId=<enter your _id>

Filter results: https://exercise-tracker-project.jylee3.repl.co/api/exercise/log?userId=<enter your _id>&from=2020-01-01&to=2020-12-30&limit=2
